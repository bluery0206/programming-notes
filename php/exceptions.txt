1. Base Exceptions
    Exception - Parent class for all exceptions. Use for generic errors.
    Error - Base class for internal PHP errors (e.g., type errors, syntax errors).
    Throwable - Interface implemented by both Exception and Error.

2. SPL Exceptions (Standard PHP Library)
    Logic Exceptions (Extend LogicException)
        LogicException              - Base for exceptions due to logical errors (e.g., incorrect method calls).
        BadFunctionCallException    - Triggered when a function doesn’t exist or isn’t callable.
        BadMethodCallException      - Thrown when a method doesn’t exist or isn’t callable.
        DomainException             - Use when a value violates domain rules (e.g., invalid business logic).
        InvalidArgumentException    - When a function/method receives an invalid argument.
        LengthException             - For invalid length issues (e.g., string/array too short/long).
        OutOfRangeException         - When an illegal index is accessed (e.g., array out of bounds).
    Runtime Exceptions (Extend RuntimeException)
        RuntimeException        - Base for failures at runtime (e.g., file not found, DB errors).
        OutOfBoundsException    - When a key/index doesn’t exist (e.g., invalid array key).
        OverflowException       - When adding to a full container (e.g., fixed-size array).
        UnderflowException      - When removing from an empty container (e.g., stack/queue).
        UnexpectedValueException- When a value doesn’t match expectations (e.g., invalid type).
3. Predefined Exceptions (Common Use Cases)
    PDOException        - Thrown by PDO for database errors.
    JsonException       - For JSON encoding/decoding failures.
    DivisionByZeroError - On division/modulo by zero.
    TypeError           - For type mismatches (e.g., wrong argument type).
    ParseError          - For syntax errors in code (e.g., eval() failures).
    AssertionError      - When assert() fails.
4. Other Notable Exceptions
    ArithmeticError         - For math errors (e.g., bit shift overflow).
    CompileError            - For compilation errors (rare in userland).
    SessionHandlerException - For session-related failures.
